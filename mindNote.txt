А тут главный контекст QueryProvider (следим за всем)

папка _auth тут храняться наши страницы входа и регистрации

папка _root тут хранятся все остальные страницы домашняя и т.д.

папка components/shared тут храняться функции вроде загрузчик и т.д. которые мы используем для отображения загрузки

папка components/ui тут будут храниться компоненты из библиотеки shad/cn такие как кнопки формы и т.д.

папка context тут храниться контекст которым мы оборачиваем наш вход и регистрация для постоянного котроля авторизированности

папка lib/appwrite тут хранять все функция по общению с back end сервисом appwrite (запись чтение и т.д.)

папка lib/react-query тут хранятьcя наше централизированное хранилище состояний где мы будем контрольровать и вызывать функции из appwrite

папка lib/validation тут хранятся библиотека zod которая нужна для проверки правильности ввода в форму

папка types тут будут храниться обекты необходимые для typescript

файл vite-env.d.ts тут храняться переменные необходимые для связи с app write
